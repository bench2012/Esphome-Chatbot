# Esphome-Chatbot
Esphome based chatbot using esp32s3, oled, i2s microphone and a speaker 

Find this on Aliexpress whichis the cheapest all-in-one ESP32S3 AI chatbot board. It basically consists of an XH-S3E, which accounting to this web site:

https://manuals.plus/sparkleiot/xh-s3e-ultra-low-power-wifi-module-manual
https://fcc.report/FCC-ID/2BCLR-XH-S3E/6776438.pdf

is just an ESP32S3-N16R8 and some peripherals; an i2s Micophone, an i2s Mono D-Amp with speaker connector an WS2812 (GPIO. It has pin out for 3.3v/GND, i2c (SDA/GPIO41 and SCL/GPIO42) , VIn (5V??), TX/RX.
